## About

Godot Scene Manager Plugin is a small workflow tool for Godot projects. It helps keep scene paths organized and exposes a central manager for switching between scenes.

## What I worked on

I worked on the editor plugin behavior, the autoload manager, and the scene-switching workflow. The goal was to make a repeated Godot task easier to reuse across projects.

## What I like about it

I like that this project is small and practical. It is not a full game, but it solves a workflow problem that comes up often when building Godot projects.
