## Overview

Godot Scene Manager Plugin is a small workflow tool for Godot projects. It helps keep scene paths organized and exposes a central manager for switching between scenes.

## My focus

The focus is practical game-development workflow: reducing scattered scene-path handling and making scene transitions easier to manage from one place.

## Why it matters

This is a useful supporting project because it shows tool-building for game projects, not just gameplay prototypes. It is small, but it solves a real workflow problem that appears repeatedly in Godot work.
